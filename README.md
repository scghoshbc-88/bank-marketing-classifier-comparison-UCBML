# bank-marketing-classifier-comparison-UCBML
The goal is to compare the performance of the classifiers (k-nearest neighbors, logistic regression, decision trees, and support vector machines) you encountered in this section of the program. You will use a dataset related to the marketing of bank products over the telephone.
# Bank Marketing Classifier Comparison

Predict whether a customer will subscribe to a term deposit based on demographic, campaign, and economic variables.

## Models Compared
- KNN
- Logistic Regression
- Decision Tree
- SVM

## Best Model
Logistic Regression

## Best ROC-AUC
0.790

## Key Findings
- Dataset is highly imbalanced.
- Logistic Regression achieved the best ROC-AUC.
- Customer and campaign attributes can effectively predict subscription likelihood.

## Recommendations
Deploy Logistic Regression as a lead-scoring tool to improve marketing efficiency and reduce unnecessary customer contacts.
